\begin{frame}[fragile]
\frametitle{Enunciado del problema}
\fontsize{16pt}{18pt}\selectfont
Asignar memoria dinamicamente al vector que contiene las direcciones de los nombres ingresados y utilizar la memoria justa para cada nombre. \\
La idea es que si reservé espacio para un vector que pueda almacenar 256 bytes pero ingreso un nombre que ocupa 5 bytes, utilicemos los recursos conocidos para que sólo se usen los espacios justos de memoria para esos 5 bytes y no los 256 bytes para cada nombre ingresado). \\
Termina el programa cuando un nombre comienza con el simbolo {\color{blue}@} 
\end{frame}

\begin{frame}[fragile]
 \frametitle{Ejemplo de {\color{yellow}malloc} y {\color{yellow}realloc} }
  \lstset{basicstyle=\tiny}
 \begin{lstlisting}
int main (void)
{
    int     i=0,j=0;
    char    c,nombres[256];
    char *  p;
    char ** adr;
    
    adr= (char **) malloc(sizeof(char *));
    do{
        fgets(nombres , 256 , stdin );          
        j = strlen(nombres);
        p = (char *) malloc (j*sizeof(char));
        strcpy ( p , nombres);
        *(adr + i) = p;     
        c = *(*(adr+i));
        if ( c != '@' ){
            adr = (char **) realloc( adr ,(i+2)* sizeof(char *)); 
            i++;
        }else{
            free(p);
            *(adr + i) = NULL;
        }      
    }while( c != '@' );
    for(i=0 ; *(adr+i) != NULL ;i++){
        printf("nombre% d :% s ",i, *(adr+i) );
        free(*(adr+i) );
    }
    free (adr);
    return 0;
}
   \end{lstlisting}
